# Yatsuiii

Rust and Go focused builder working on causal debugging, observability, and applied ML systems.

## Current Focus

- Developer tooling for API workflow validation and trace diffing
- Cost and spend attribution systems for cloud and LLM workloads
- ML systems with an interest in robustness across real deployment settings

## Featured Repositories

- [api-causality-engine](https://github.com/Yatsuiii/api-causality-engine) - Stateful API workflow testing and trace diffing in Rust
- [llmtrace](https://github.com/Yatsuiii/llmtrace) - Self-hosted LLM proxy with anomaly detection and deploy-to-spend attribution
- [costtrace](https://github.com/Yatsuiii/costtrace) - AWS cost anomaly attribution using CloudTrail and GitHub Actions deploy evidence
- [Brain-Connectivity-GCN](https://github.com/Yatsuiii/Brain-Connectivity-GCN) - Cross-site adversarial GCN for ASD detection from resting-state fMRI
- [fractured_perception](https://github.com/Yatsuiii/fractured_perception) - Experimental Rust game prototype built around asymmetric perception
- [naruto-portfolio](https://github.com/Yatsuiii/naruto-portfolio) - Themed portfolio built with Next.js and TypeScript
